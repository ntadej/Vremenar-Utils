A collection of utilities for the [Vremenar API](https://github.com/ntadej/Vremenar-API).

## Installation and running

### Python PDM

This project uses [PDM](https://pdm.fming.dev) to track dependencies.
For basic setup run

```shell
pdm sync
```

## Contributing

### pre-commit

This project uses `pre-commit`. To setup, run

```shell
pre-commit install
```

To check all files run

```shell
pre-commit run --all
```
